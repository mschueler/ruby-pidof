ruby-pidof
==========

Gem to find the pid of a process. Tested to work on OS X and Linux.

Port of node-pidof (https://github.com/calmh/node-pidof).


Installation
------------

  gem install pidof



Example
-------

  require 'pidof'
  pid = Pidof.find('login')
